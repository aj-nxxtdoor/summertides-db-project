# SQL Sprint Project

## Learning Outcomes

By completing this sprint you should be able to:

- Create databases
- Create tables
- Define primary keys
- Define foreign keys
- Apply constraints
- Insert data
- Retrieve records
- Filter records
- Sort data
- Limit results
- Use CASE expressions
- Aggregate data
- Group records
- Join tables
- Create Views
